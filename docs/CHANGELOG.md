# 更新日志填写指南

登录 Admin 后台 → 左侧菜单「更新日志」→ 逐条添加以下版本。

添加时注意：

- **版本号**：填写 `版本` 列
- **日期**：填写 `日期` 列（格式 YYYY-MM-DD）
- **内容**：填写 `内容` 列，**每行一条更新，用换行分隔**（Admin 后台 textarea 中直接回车换行即可）
- **顺序**：最新版本在此文件中应写在最前
